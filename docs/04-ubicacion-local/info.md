# Ubicación y Local

:::info[Estado del documento]
🔄 Documento base completo. Actualizar cuando se identifiquen opciones concretas de local en Fusagasugá.
:::

:::tip[Ventaja de Fusagasugá]
A diferencia de Bogotá o Medellín, Fusagasugá aún no tiene saturación de hamburgueserías fast casual de calidad. Eso significa que la ubicación correcta no necesita ser la más costosa — necesita ser la más estratégica.
:::

---

## Perfil del local ideal para Entre Hojas

Antes de salir a buscar, tener claro qué se busca evita perder tiempo evaluando opciones que no encajan.

| Atributo | Requerimiento |
|---|---|
| **Área mínima techada** | 50 – 65 m² |
| **Espacio exterior / terraza** | Sí — indispensable para el concepto |
| **Área total ideal** | 80 – 100 m² (techado + exterior) |
| **Tipo de local** | Planta baja con fachada visible desde la vía |
| **Extracción de humos** | Debe permitir ducto hacia exterior o azotea |
| **Capacidad eléctrica** | Mínimo 30 amperios trifásicos |
| **Aforo interior** | 20 – 25 personas |
| **Aforo exterior / terraza** | 10 – 15 personas adicionales |
| **Aforo total** | 30 – 40 personas |
| **Arrendamiento estimado** | $2.000.000 – $3.500.000/mes |
| **Modalidad** | Arriendo desde cero |

## Distribución interna recomendada

Para un local de 50–65 m² techados con terraza exterior, la distribución óptima es:

```
┌─────────────────────────────────────────┐
│  TERRAZA / ESPACIO VERDE EXTERIOR       │
│  ~25–35 m²                              │
│  Mesas con plantas, vegetación,         │
│  iluminación cálida                     │
└────────────────┬────────────────────────┘
                 │  Acceso
┌────────────────▼────────────────────────┐
│  SALÓN INTERIOR          │   COCINA     │
│  ~30–35 m²               │   ~15–18 m² │
│  Mesas y sillas          │             │
│  Mostrador / caja        │  Plancha    │
│                          │  Freidora   │
│                          │  Mesa fría  │
│                          │  Nevera     │
└──────────────────────────┴─────────────┘
│  BAÑOS  │
│  ~5 m²  │
└─────────┘
```

:::note
La cocina debe estar contigua al mostrador de caja para que el flujo pedido → preparación → entrega sea directo. La terraza debe ser visible desde el interior — parte del atractivo es que quien está adentro vea el ambiente verde exterior.
:::

## Zonas de Fusagasugá a evaluar

Fusagasugá tiene varias zonas con características distintas. El concepto de Entre Hojas encaja mejor en unas que en otras. Evaluar cada zona antes de buscar local activamente.

### Zona 1 — Centro histórico y área comercial principal
**Calles 6–10 con carreras 6–10 aproximadamente**

| Factor | Evaluación |
|---|---|
| Tráfico peatonal | ⭐⭐⭐⭐⭐ Muy alto |
| Perfil del público | Mixto — familias locales, compradores, paso rápido |
| Arriendo estimado | Alto — $2.800.000 – $4.000.000 |
| Competencia | Media — locales de comida rápida tradicional |
| Espacio exterior disponible | Difícil — locales típicamente sin terraza |
| Compatibilidad con el concepto | ⭐⭐⭐ Media |

**Análisis:** Alto tráfico pero difícil conseguir espacio con terraza. El público de paso rápido no siempre encaja con un ambiente donde "quedarse" es la propuesta. Evaluar solo si aparece un local con patio interior o espacio exterior.

---

### Zona 2 — Avenida Colón y corredores de acceso principal
**Vías principales de entrada y salida de la ciudad**

| Factor | Evaluación |
|---|---|
| Tráfico vehicular | ⭐⭐⭐⭐⭐ Muy alto |
| Perfil del público | Turistas bogotanos, viajeros, familias en tránsito |
| Arriendo estimado | Medio — $2.000.000 – $3.200.000 |
| Competencia | Baja en fast casual de calidad |
| Espacio exterior disponible | Posible — locales más amplios |
| Compatibilidad con el concepto | ⭐⭐⭐⭐ Alta |

**Análisis:** Excelente para capturar turistas bogotanos que llegan o salen de Fusagasugá. El tráfico del fin de semana es muy alto. Buscar locales sobre vía principal con estacionamiento cercano y posibilidad de terraza lateral o frontal.

---

### Zona 3 — Barrios residenciales en expansión (norte y sur)
**Sectores como Santafé, Los Comuneros, urbanizaciones nuevas**

| Factor | Evaluación |
|---|---|
| Tráfico peatonal | ⭐⭐ Medio-bajo entre semana |
| Perfil del público | Familias residentes, estrato 3–4 |
| Arriendo estimado | Bajo — $1.500.000 – $2.500.000 |
| Competencia | Muy baja |
| Espacio exterior disponible | Alto — locales más amplios y económicos |
| Compatibilidad con el concepto | ⭐⭐⭐⭐ Alta para familias |

**Análisis:** Arriendo más bajo y más posibilidad de terraza, pero el tráfico entre semana es insuficiente sin una base de clientes habituales. Funciona mejor como segunda sede cuando el negocio ya esté consolidado. **No recomendado para la primera ubicación.**

---

### Zona 4 — Zona rosa y de entretenimiento
**Sector de bares, restaurantes y vida nocturna de Fusagasugá**

| Factor | Evaluación |
|---|---|
| Tráfico peatonal | ⭐⭐⭐⭐ Alto fines de semana |
| Perfil del público | Jóvenes, universitarios, turistas nocturnos |
| Arriendo estimado | Medio — $2.000.000 – $3.500.000 |
| Competencia | Media — restaurantes variados |
| Espacio exterior disponible | Posible |
| Compatibilidad con el concepto | ⭐⭐⭐⭐ Alta para jóvenes |

**Análisis:** Muy buena opción si el horario de operación incluye noches y fines de semana. El público joven valora el ambiente diferenciado. Tráfico entre semana más bajo. Evaluar con visitas en diferentes horarios.

---

### Zona 5 — Cerca de colegios y universidades
**Sector UNIMINUTO, colegios privados, zona estudiantil**

| Factor | Evaluación |
|---|---|
| Tráfico peatonal | ⭐⭐⭐⭐ Alto en horario escolar/universitario |
| Perfil del público | Estudiantes, profesores, padres de familia |
| Arriendo estimado | Medio — $1.800.000 – $3.000.000 |
| Competencia | Media — comida económica predomina |
| Espacio exterior disponible | Posible |
| Compatibilidad con el concepto | ⭐⭐⭐ Media |

**Análisis:** Buen volumen entre semana pero ticket promedio más bajo. El público estudiantil es sensible al precio. Encaja si el menú tiene opciones en el rango bajo del ticket ($28.000–$32.000). Evaluar según confirmación del segmento universitario del módulo 1.

## Zonas recomendadas como prioridad

Según el concepto de Entre Hojas — ambiente verde, turistas bogotanos, familias, fast casual — el orden de prioridad para buscar es:

```
1° Avenida Colón / corredor de acceso principal
   → Mejor captura de turistas, tráfico constante, posibilidad de terraza

2° Zona rosa / entretenimiento
   → Público joven y familias fines de semana, ambiente compatible

3° Centro (solo si aparece local con terraza o patio)
   → Alto tráfico pero difícil conseguir el espacio ideal
```

## Cómo evaluar cada local candidato

Cuando encuentren opciones concretas, usar esta metodología antes de decidir:

### Paso 1 — Visita de campo (sin ver el local aún)

Ir a la zona en tres momentos distintos y anotar:

| Momento | Qué observar |
|---|---|
| Lunes o martes al mediodía | Tráfico real entre semana |
| Viernes en la noche | Tráfico nocturno y tipo de público |
| Sábado o domingo al mediodía | Tráfico de turistas y familias |

Contar las personas que pasan frente al local en 30 minutos. Un local viable debe tener mínimo **80–100 personas/hora** en horario pico.

### Paso 2 — Verificación técnica del local

Antes de negociar el arriendo, verificar con un técnico o contratista:

- [ ] **Extracción de humos:** ¿hay ruta viable para el ducto hacia exterior o azotea?
- [ ] **Capacidad eléctrica:** revisar el tablero — necesitan mínimo 30A trifásicos
- [ ] **Presión de agua:** abrir todos los grifos simultáneamente
- [ ] **Desagüe:** capacidad para trampa de grasas
- [ ] **Espacio exterior:** ¿es de uso exclusivo del local? ¿requiere permiso adicional?
- [ ] **Estado de pisos y paredes:** ¿requieren revestimiento para norma sanitaria?

:::caution
La verificación técnica debe hacerla un contratista o técnico, no el arrendador. Cuesta $80.000–$150.000 y puede ahorrar $10.000.000 en sorpresas.
:::

### Paso 3 — Verificación legal

- [ ] Consultar uso de suelo en Planeación Municipal de Fusagasugá (antes de firmar)
- [ ] Verificar que el arrendador sea el propietario real (pedir certificado de tradición y libertad)
- [ ] Confirmar que no hay deudas de servicios públicos pendientes sobre el inmueble

### Paso 4 — Negociación del contrato

**Puntos clave a negociar:**

| Punto | Lo mínimo aceptable |
|---|---|
| **Duración** | 2 años con opción de renovación |
| **Período de gracia** | 1–2 meses sin arriendo para adecuaciones |
| **Cláusula de obras** | Definir por escrito qué mejoras se pueden hacer y qué pasa con ellas al terminar |
| **Incremento anual** | Máximo IPC del año anterior |
| **Subarrendamiento** | No aplica — pero incluir cláusula de cesión si la SAS necesita restructurarse |

:::danger[No firmar sin periodo de gracia]
La adecuación del local toma entre 3 y 6 semanas. Pagar arriendo mientras se adecua sin vender nada consume capital de trabajo innecesariamente. La mayoría de arrendadores de locales vacíos acepta 30–45 días de gracia.
:::

## Regla financiera del arriendo

El arriendo no debe superar el **10% de las ventas mensuales proyectadas**.

| Escenario de ventas | Arriendo máximo recomendado |
|---|---|
| Conservador ($31.850.000/mes) | $3.185.000 |
| Pesimista ($18.200.000/mes) | $1.820.000 |

Si el local que les gusta cuesta más de $3.200.000, deben proyectar ventas que lo justifiquen o negociar el canon a la baja.

## El espacio exterior — cómo aprovecharlo para el concepto

La terraza o espacio al aire libre es uno de los mayores diferenciadores de Entre Hojas en Fusagasugá. Algunas consideraciones para aprovecharlo bien:

**Elementos que refuerzan el concepto verde:**
- Plantas en materas grandes (helechos, pothos, palmeras pequeñas)
- Estructura de madera o guadua para delimitar el espacio
- Iluminación cálida con bombillas tipo Edison o luces colgantes
- Piso en deck de madera o gravilla decorativa
- Pérgola o toldo para proteger del sol y la lluvia

**Consideraciones prácticas:**
- El espacio exterior debe tener cubierta parcial — Fusagasugá tiene lluvias frecuentes y sin techo los clientes no pueden quedarse
- Verificar que el arrendador permita modificaciones en el espacio exterior
- La terraza debe ser visible desde la vía — es el gancho visual que atrae al cliente que pasa

**Capacidad y distribución exterior sugerida:**
- 3–4 mesas de 2–4 personas cada una
- Mínimo 1,20 m entre mesas para circulación cómoda
- Zona de plantas como separador visual entre mesas y la vía pública

## Checklist de decisión final

Antes de firmar el contrato de arrendamiento, verificar que se cumplan todos estos puntos:

**Técnicos — todos obligatorios:**
- [ ] Permite instalación de campana extractora con ducto al exterior
- [ ] Capacidad eléctrica suficiente (mínimo 30A trifásicos)
- [ ] Espacio exterior de uso exclusivo con posibilidad de cubierta
- [ ] Desagüe con capacidad para trampa de grasas
- [ ] Uso de suelo verificado en Planeación Municipal

**Financieros — todos obligatorios:**
- [ ] Arriendo ≤ 10% de ventas proyectadas conservadoras
- [ ] Período de gracia negociado (mínimo 30 días)
- [ ] Contrato mínimo 2 años con opción de renovación
- [ ] Cláusula de obras definida por escrito

**Estratégicos — mínimo 3 de 4:**
- [ ] Tráfico peatonal/vehicular verificado en campo: ≥ 80 personas/hora en pico
- [ ] Competencia directa en radio de 300 m: ninguna hamburguesería fast casual similar
- [ ] Parqueadero a menos de 100 m
- [ ] Visibilidad de fachada desde la vía principal
